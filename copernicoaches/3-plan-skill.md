# Skills de Planificación Arquitectónica

## analyze_requirements
Analiza especificaciones funcionales y documentos de arquitectura para identificar componentes clave y dependencias.

## define_implementation_steps
Define un plan de implementación de máximo 5 pasos, cada uno con objetivos, tareas, riesgos y entregables.

## generate_step_files
Genera archivos .md individuales por cada paso, con nombres como SPEC01-xxxx.md, incluyendo título, descripción, dependencias y mitigaciones.

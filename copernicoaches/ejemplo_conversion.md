# Demostración de Conversión de Documentos

## Introducción

Este es un documento de ejemplo para demostrar la conversión entre formatos Markdown y Word usando **jano**.

### Características principales

- **Encabezados**: Se preservan los niveles de encabezado
- **Listas**: Tanto ordenadas como desordenadas se mantienen
- **Formatos**: *Cursiva*, **negrita**, y más
- **Enlaces**: [Copérnico](https://copernico.es)

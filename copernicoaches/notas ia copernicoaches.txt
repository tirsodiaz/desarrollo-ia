1.instalar librería openspec en nodejs -> npm install -g @fission-ai/openspec@latest
2.Ir a la carpeta del proyecto que queremos usar -> cd c:\cursos\desarrollo-ia\copernicoaches

Reiniciar VS Code para que los slash commands queden activos en el chat de Copilot: 
	/opsx:propose "tu idea" — proponer un cambio
	/opsx:spec — gestionar especificaciones
	/opsx:change — gestionar cambios
si no disponibles en el chat: El problema está claro: los archivos opsx-*.prompt.md están en prompts, 
pero VS Code solo busca prompts en la raíz del workspace: prompts. 
Opciones para solucionarlo: 
A) Configurar chat.promptFilesLocations en VS Code settings para que incluya esa ruta adicional (sin mover los archivos).
B) Mover/copiar los archivos opsx-*.prompt.md a prompts.
Opcion A)
Crear C:\cursos\desarrollo-ia\copernicoaches\.vscode\settings.json con:
{
  "chat.promptFilesLocations": {
    ".github/prompts": true,
    "explorer/.github/prompts": true
  },
  "chat.instructionsFilesLocations": {
    ".github/skills": true,
    "explorer/.github/skills": true
  }
}
Opcion B) (por defecto se crean en raíz)
{
  "chat.promptFilesLocations": {
    ".github/prompts": true,
    ".github/skills": true
  }
}

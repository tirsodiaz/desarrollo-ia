# ¿Qué es un LLM?

## Introducción

Un **LLM (Large Language Model)** o *Modelo de Lenguaje de Gran Tamaño* es un sistema de inteligencia artificial diseñado para comprender y generar lenguaje natural.

Estos modelos se entrenan con grandes cantidades de texto y son capaces de realizar tareas como:

- Responder preguntas

- Resumir información

- Traducir idiomas

- Generar contenido

## Concepto Básico

Un LLM funciona prediciendo la siguiente palabra (o token) en una secuencia, basándose en el contexto previo.

### Ejemplo simple

Entrada: "El cielo es de color..."
Salida probable: "azul"

Este mecanismo, aunque simple en apariencia, permite generar textos complejos y coherentes.

## Características Principales

### 1. Gran volumen de datos

Los LLM se entrenan con enormes cantidades de texto provenientes de libros, páginas web y otros documentos.

### 2. Arquitectura basada en Transformers

Utilizan una arquitectura llamada **Transformer**, que permite procesar relaciones entre palabras en una oración.

### 3. Capacidad de generalización

Pueden adaptarse a múltiples tareas sin necesidad de entrenamiento específico para cada una.

## Estructura de un LLM

| Componente | Descripción |
| --- | --- |
| Datos de entrenamiento | Texto utilizado para entrenar el modelo |
| Tokens | Unidades básicas (palabras o fragmentos) |
| Parámetros | Variables internas del modelo (millones o billones) |
| Contexto | Información previa utilizada para generar texto |

1. Mantener los niveles de encabezado (H1, H2, H3)

4. Evitar pérdida de información

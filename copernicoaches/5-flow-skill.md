# Skills de Implementación de Código

## translate_plan_to_code
Analiza plan de implementación y tareas, traduce a código funcional respetando frameworks y dependencias.

## generate_modular_code
Genera código limpio, modular, documentado y con buenas prácticas del lenguaje objetivo.

## validate_against_plan
Verifica que el código generado cumple exactamente lo especificado en el plan sin añadidos.
